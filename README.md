# Weather-Inquire
# 介绍
本项目为一个简单的天气查询网页，主要是作为初次使用 Boopstrap 的一个练手项目，但可惜的是理论上应该有的响应式还是被写的乱七八糟😞

# 采用技术
**HTML + CSS + JS + VUE + Bootstrap + Axios**
